**Teaser:** *"Am Ende zeigen wir Ihnen, wie sich das im Vergleich zu anderen Angeboten auf dem Markt schlägt."*

## Section 2: WIE — Dolmetscher (~2 min)
*David scrollt zu: "Ihr Dolmetscher für Daten"*

- "Ihre Daten sprechen verschiedene Sprachen. Die KI wird zum Dolmetscher — wir liefern die Wörterbücher."
- 4 Connector-Typen kurz zeigen (Telefonzentrale)

## Section 3: WANN — Setup Plan (~1.5 min)
*David scrollt zu: "Fünf Tage. Fertig."*

- **Tag 1:** Box online, erstes Modell läuft
- **Tage 2–3:** Hier passiert die eigentliche Arbeit — Datenquellen anbinden. SQL, Confluence, Dateiserver, AS/400. Die Konnektoren aus Section 2 werden hier eingerichtet.
- **Tag 4:** Mit echten Daten getestet — erst intern, dann Demo
- **Tag 5:** Übergeben, Team geschult. Läuft.

**Referenz:** "Ein Kunde aus der Fertigungsindustrie — 294 Seiten: 5 Tage auf Power 10, 13 Minuten auf DGX Spark."

## Section 4: WARUM — Labor + Startrampe (~1.5 min)

## Section 5: VERGLEICH — Comparison (~2 min)
*David scrollt zu: "Drei Wege zur KI."*

**Vorgehen:**
1. "So schlägt sich das im Vergleich." → **Pause, 15–20 Sek. lesen lassen.**
2. "Haben Sie dazu Fragen?" → Auf Reaktionen warten.
3. Wenn Fragen: gezielt die Zeile beantworten.
4. Wenn Stille: 2–3 stärkste Zeilen ansprechen.

**Back-Pocket One-Liners (nur bei Bedarf):**

| Zeile | Was sagen |
|-------|-----------|
| Preis | "5 Nutzer, Max-Plan: €5.500 pro Jahr — und Sie mieten nur. Bei uns: €10K einmalig, Sie besitzen es." |
| Datenschutz | "Cloud: US-Server, CLOUD Act. Bei uns: lokal, DSGVO. Der Unterschied: für €10K, nicht €250K." |
| Chat UI | "Alle drei haben es — bei Spire müssen Sie es selbst bauen." |
| MCP / APIs | "Claude hat's — aber Sie konfigurieren alleine. Wir binden es an." |
| Datenbanken | "Claude: gar nicht. Wir: eingerichtet. Spire: möglich, aber IBM-Consulting-Preise." |
| Dokumentensuche | "Claude: Datei-Upload, temporär. Bei uns: Vector DB — dauerhafte semantische Suche." |
| Skills | "Firmenwissen als Anweisungen — Ihre Prozesse, Ihre Logik. Bauen wir für Sie." |
| Vision | "Beide haben es, bei uns ist es konfiguriert." |

| Admin/SSO | "Claude: ab Team-Plan, Aufpreis. Bei uns: volle Kontrolle." |
